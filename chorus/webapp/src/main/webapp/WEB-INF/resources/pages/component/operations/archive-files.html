The selected files will be put into the archive. It allows to reduce the storage costs. However, the file download are not immediate in this case, since it takes several hours to get the data from archive.

Archive operation is not available for all selected files.

Possible reasons:

  • Some of the selected files are already archived
  • Some of the selected files are used in experiments
  • Some of the selected files are moving on Archive Storage now
  • For the some selected files laboratory Archiving feature was disabled
  • Some of selected files are corrupted
